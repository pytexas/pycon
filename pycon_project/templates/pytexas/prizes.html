Prizes

Each attendee at PyTexas will be given a set of printed tickets to divvy up among the prize drawing boxes, with one box for each prize, thereby allowing each attendee to weighting the odds toward prizes of particular interest.

The following print books will be available as prizes from No Starch:

  • The Book of CSS3
  • Eloquent JavaScript
  • Gray Hat Python
  • Learn You a Haskell for Great Good!
  • The Essential Blender
  • Code Craft
  • The Manga Guide to Databases

No Starch is also sending along some cool swag in the form of t-shirts and shot glasses. Those will be made available on the swag table on a first come, first serve basis.
